## Dependencies

- React
- React-dom
- React-router-dom
- React-scripts
- Auth0
- Material-react-table
- React-chartjs-2
- React-toggle-dark-mode
- Tailwindcss
- Web-vitals
- Axios
- Node.js
- Express
- Cors
- Dotenv
- PG

## Setup

Install dependencies with `npm install` in each respective `/client` and `/server`.

## [Frontend] Running Webpack Development Server

```sh
cd client
npm start
```

## [Backend] Running Backend Servier

```sh
cd server
npm run local
```
